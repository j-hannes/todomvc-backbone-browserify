todomvc-backbone-browserify

requirements
------------

* `npm install -g gulp watchify`
* [Chrome LiveReload](https://chrome.google.com/webstore/detail/livereload/jnihajbhpnppcggbcgedagnkighmdlei)

install
-------

* `npm install`

run
---

* `gulp live`
